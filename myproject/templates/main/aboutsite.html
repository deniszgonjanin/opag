This website was authored using nothing but free GNU and Linux tools. While GNU and Linux are not directly related to Python, the language was born of the same principles--it is open-source, and freely available, costing nothing for the public to download and use. In fact, it shows no discrimination towards platforms, as it is freely available for Unix, Windows, MacOS, even PalmOS. This is the kind of quality and flexibility created by open-source.

All HTML code for the site was edited using the Vim editor on Linux. The images were created with the Gimp, with tweaks and conversions by ImageMagick. We hope that the site stands as an example that the best tools are not necessarily the most expensive ones.

The website itself is hosted on a Linux virtual server, provided by Digitaltorque Consulting Inc., via Webfaction. The web server is Apache, and the site is run by Django, which is Python Powered.

The site is version controlled with Git, and published using Github. Feel free to clone it and send me patches.

If you have any problems, please inform me.
